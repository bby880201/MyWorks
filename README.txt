This application is a Simple abc file parsing and playing application. 

How to run the application:

Run the controller.controller class, or directly run the entire project in eclipse.
The application window will pop up. 

Type in an abc file name and click on "load" button. The abc will be loaded and the contents will be displayed below. 
Click on "parse" button, the loaded file will be parsed and the structure will be displayed below.
Select an instrument from the drop list and click on "play" button, the parsed file will be played.
Before music ends, "stop" button can stop the music immediately.
